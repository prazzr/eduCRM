EduCRM System Walkthrough

This guide explains how to use the newly implemented CRM, LMS, and Accounting system.

1. Getting Started

  • URL: http://localhost/CRM/
  • Default Admin: admin@example.com / password

2. Core Modules

CRM (Student Management)

Goal: Manage student inquiries and onboard them.

  1. Add Inquiry: Go to Inquiries > New Inquiry. Fill in details like "Intended Country" and "Course" (IELTS, PTE).
  2. Convert to Student: On the Inquiry List, click Convert to Student. This creates a User account for them (Default password: password123).

LMS (Learning Management)

Goal: Manage classes and assignments.

  1. Create Course: Go to LMS > Manage Courses (e.g., add "IELTS").
  2. Create Class: Go to LMS > Manage Classes. Create a class (e.g., "IELTS Morning Setup") and assign a Teacher.
  3. Classroom View: Click View on a class.
    • Enroll Students: Select a student from the dropdown to add them to the class.
    • Post Material: Teachers can upload notices, reading materials, or assignments.
    • Submissions: Students log in, see the assignment, and upload their work.

Accounting (Fees & Payments)

Goal: Track student finances.

  1. Fee Types: Go to Accounting > Manage Fee Structure to define fees (e.g., "Tuition Fee $1000").
  2. Assign Fee: Go to Accounting (Ledger List), click a student. In the "Assign Fee" box, select a Fee Type and Amount.
  3. Record Payment: On the same Student Ledger page, select an unpaid invoice and record a payment (Cash/Bank). The system tracks partial payments and updates the status to "Paid" automatically.

3. Roles & Permissions

  • Admin: Full access.
  • Counselor: Can manage Inquiries.
  • Teacher: Can manage their assigned Classes and view Student Submissions.
  • Student: Can view their Classes, Submit Assignments, and view their Financial Ledger.
