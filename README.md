## Quick setup
- Copy `config.sample.php` to `config.php` and fill in DB credentials.
- Import `schema.sql` or `database_dump.sql` to create the database schema.
- Place this project in a PHP-capable web server (e.g. XAMPP) and visit `http://localhost/CRM`.
