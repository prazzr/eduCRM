### Adding a New Inquiry

1. Click **"Inquiries"** in the menu
2. Click **"Add Inquiry"** button
3. Fill in the form:
   - Name, Email, Phone (required)
   - Source (website, referral, etc.)
   - Program of interest
   - Notes
4. Click **"Save"**

## 5. Scheduling Appointments

### Book an Appointment

1. Click **"Appointments"** in menu
2. Click **"Add Appointment"**
3. Fill in:
   - Title and description
   - Date and time
   - Duration
   - Attendees
   - Location/Meeting link
4. Click **"Save"**

### Calendar View

- Click **"Calendar"** to see all appointments
- Click on any appointment to view/edit
- Drag & drop to reschedule (if enabled)

---

## 6. Sending Messages

### SMS/WhatsApp/Viber

1. Go to **"Messaging"** in menu
2. Click **"Send Message"**
3. Select:
   - Channel (SMS, WhatsApp, Viber)
   - Recipients (individual or bulk)
   - Message template or custom text
4. Click **"Send"**

### Message Templates

- Use pre-defined templates for common messages
- Personalize with student name, program, etc.
- Track delivery status

## 7. Generating Reports

### Quick Reports

1. Click **"Reports"** in menu
2. Select report type:
   - Inquiry Report
   - Student Report
   - Financial Report
   - Counselor Performance
3. Choose date range
4. Click **"Generate Report"**
5. Export as CSV, Excel, or PDF

## 9. Common Tasks

### Converting Inquiry to Student

1. Open inquiry details
2. Click **"Convert to Student"**
3. Fill in additional student information
4. Click **"Save"**

### Managing Documents

1. Go to student profile
2. Click **"Documents"** tab
3. Upload required documents
4. Track document status

### Bulk Actions

1. Go to Inquiries/Tasks/Appointments list
2. Select multiple items (checkboxes)
3. Choose bulk action:
   - Delete
   - Update status
   - Assign to user
   - Send message
4. Click **"Apply"**
