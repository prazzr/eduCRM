## Recording Specifications

### Technical Requirements:
- **Resolution:** 1920x1080 (1080p)
- **Frame Rate:** 30 fps
- **Audio:** Clear voice, no background noise
- **Format:** MP4 (H.264)
- **Length:** 5-12 minutes per video

### Software Recommendations:
- **Screen Recording:** OBS Studio, Camtasia, or ScreenFlow
- **Video Editing:** DaVinci Resolve, Adobe Premiere, or Camtasia
- **Audio:** Audacity for audio cleanup

### Recording Tips:
1. Use a clean test environment
2. Prepare script beforehand
3. Use cursor highlighting
4. Speak clearly and slowly
5. Add captions/subtitles
6. Include intro/outro
7. Add chapter markers
8. Test audio levels

## Alternative: Interactive Walkthroughs

If video creation is not feasible, use interactive walkthroughs:

### Tools:
- **Intro.js** - Step-by-step guides
- **Shepherd.js** - User tours
- **Appcues** - In-app guidance

### Implementation:
```html
<!-- Add to header.php -->
<script src="https://cdn.jsdelivr.net/npm/intro.js@5.1.0/intro.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/intro.js@5.1.0/minified/introjs.min.css">

<script>
// Start tour
function startTour() {
    introJs().setOptions({
        steps: [
            {
                element: '#dashboard',
                intro: 'Welcome to your dashboard!'
            },
            {
                element: '#inquiries-menu',
                intro: 'Click here to manage inquiries'
            }
        ]
    }).start();
}
</script>
```
