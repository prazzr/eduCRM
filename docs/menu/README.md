# EduCRM Menu Documentation

This directory contains detailed analysis and documentation for each module in the EduCRM system.

## Core Modules
- [**Dashboard**](dashboard.md) - The central landing page with role-specific widgets and analytics.
- [**Tasks**](tasks.md) - Task management with Kanban-style features and filtering.
- [**Appointments**](appointments.md) - Scheduling system for meetings and counseling sessions.

## CRM & Sales
- [**Inquiries**](inquiries.md) - Lead management with scoring and pipeline stages.
- [**Students**](students.md) - Directory of enrolled students and their profiles.
- [**Applications**](applications.md) - Tracker for university applications.
- [**Visa Tracking**](visa_tracking.md) - Workflow system for visa processing SLAs.

## Learning Management (LMS)
- [**Courses**](courses.md) - Master catalog of educational products.
- [**Classes**](classes.md) - Management of active batches and enrollments.

## Communication & Content
- [**Messaging**](messaging.md) - Gateway configuration for SMS, WhatsApp, and Viber.
- [**Templates**](templates.md) - Centralized message templates with variable support.
- [**Documents**](documents.md) - System-wide file repository and version control.
- [**Automation**](automation.md) - Workflow builder for triggered actions.

## Administration & Finance
- [**Accounting**](accounting.md) - Student ledgers, invoices, and fee types.
- [**Reports**](reports.md) - Analytics dashboards and performance metrics.
- [**Users**](users.md) - Internal staff management and role assignment.
- [**Branches**](branches.md) - Multi-location office management.
